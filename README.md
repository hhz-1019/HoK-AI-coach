# 王者荣耀BP模拟器

基于 Python (FastAPI)、Vue 3 和 ECharts 构建的王者荣耀 BP (Ban/Pick) 流程模拟与阵容推荐系统。

## 功能介绍

* **流程模拟**：还原真实的 10 Ban 10 Pick 交替选人流程，支持为英雄指定对抗路、打野、中路、发育路、游走等分路。
* **智能推荐**：根据当前双方阵容，结合英雄基础强度、克制关系、团队配合度计算动态权重，实时给出选人推荐。
* **个性化偏好**：支持在前端设置个人的英雄池（擅长/不擅长），系统将根据偏好调整推荐结果的优先级。
* **阵容战力分析**：BP 环节结束后，自动展示双方分路对位的优劣势比，并生成团队五维属性（爆发、输出、生存、控制、机动）雷达图对比。

## 使用方法

### 1. 启动后端服务

确保本地已安装 Python 环境。首先，在项目根目录下安装所需依赖：

```bash
pip install -r requirements.txt
```

然后启动 FastAPI 后端服务：

```bash
python main.py
```
*(后端服务默认运行在 http://127.0.0.1:8000)*

### 2. 运行前端页面

由于前端请求了后端接口和本地的数据文件，建议通过本地服务器环境运行前端：

* 推荐使用 VS Code 编辑器，并安装 **Live Server** 插件。
* 在项目根目录的 `index.html` 文件上右键，选择 **"Open with Live Server"**。
* 浏览器会自动打开页面，即可开始体验 BP 流程。
