# 王者荣耀 BP 智能教练大脑 (HoK AI Coach)

基于 Python (FastAPI) + Vue 3 + ECharts 构建的王者荣耀排位/赛事 BP (Ban/Pick) 战术推演与智能推荐系统。

## 🌟 核心功能

* **沉浸式黑金 UI**：电竞级视觉体验，支持分路智能推断与手动调整。
* **智能推荐引擎**：根据当前双方阵容，结合英雄基础胜率、克制链矩阵、团队配合度，进行三段式动态权重打分，实时计算 10 大最优解。
* **专属绝活档案**：支持在前端录入个人玩家的“绝活”与“雷区”英雄，算法会在推演时赋予绝活英雄最高优先级权重。
* **战力深度剖析**：10 手 BP 结束后，自动生成「各分路对位压制率推演条」与红蓝双分离的「团队五维属性（爆发、输出、生存、控制、机动）雷达图」。

## 💻 本地运行指南

本项目的后端服务已整合前端页面托管，本地运行非常简单，无需额外配置本地 Web 服务器。

### 1. 环境准备
请确保已安装 Python 3.8+ 环境。在项目根目录下打开终端，安装依赖包：

```
pip install -r requirements.txt
```

### 2. 启动服务
在终端运行以下命令启动推荐算法引擎与 Web 服务：

```
python main.py
```

### 3. 开始推演
打开任意浏览器，在地址栏输入以下地址即可进入系统：

```
http://127.0.0.1:8000
```

## 📂 项目结构

* `main.py`: FastAPI 后端主程序（包含推荐算法引擎、动态权重打分逻辑与 Web 路由）。
* `index.html`: Vue 3 + Tailwind CSS 构建的前端交互界面。
* `hero_database_full.json`: 英雄基础属性与相互克制/配合关系网数据库。
* `requirements.txt`: Python 运行依赖清单。
